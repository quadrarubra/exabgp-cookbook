# exabgp cookbook

Installs and configures [ExaBGP](https://github.com/Exa-Networks/exabgp), the swiss-army knife of networking.

## Supported Platforms

* Ubuntu 12.04
* Ubuntu 13.04
* Ubuntu 14.04

## Dependencies

* python
* apt
* runit
* git (if installing latest or from source)
